Política de Privacidade

Esta é uma versão modelo de Política de Privacidade para o site Controle Digital. Aqui explicamos quais dados coletamos, como são usados e os direitos dos usuários, em conformidade com a LGPD.

Dados que podemos coletar

  • Informações que você fornece voluntariamente (nome, e-mail)
  • Dados de uso armazenados localmente (preferências, estatísticas de sessões)

Uso dos dados

Usamos os dados para personalizar a experiência (ex.: lembrar preferências do timer) e melhorar o serviço. Nenhum dado sensível é coletado por padrão.

Direitos do usuário

Você pode solicitar acesso, correção ou exclusão dos seus dados salvos localmente. Para dados armazenados apenas no seu navegador (localStorage), use as opções do próprio site para apagar seu perfil.

Contato

Para dúvidas sobre privacidade, entre em contato: suporte@controdigital.example (endereço modelo).
